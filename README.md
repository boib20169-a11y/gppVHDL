As documented in our project report, actual COE 328 Lab 6.pdf, my partner and I set out to take everything we learned about digital logic and apply it to building a simple, general-purpose processor from the ground up. Our objective was to create a modular 8-bit execution system capable of seamlessly translating algorithmic states into precise hardware instructions using microcoded control. To achieve this, we developed the architecture using VHDL and structural block diagrams, incorporating dual input latches to serve as registers, a Moore-type Finite State Machine to manage state transitions, and a 4-to-16 decoder to generate 16 unique operational microcodes. These control lines directed a custom Arithmetic Logic Unit that we progressively built up to execute everything from fundamental arithmetic and bit-level rotations to conditional comparisons hooked up to a 7-segment display. Ultimately, after navigating real-world hardware headaches like pinpointing board pin assignments and grounding floating pins during testing, we validated our entire design through waveform simulations to deliver a fully functional and verified microcoded processing engine.
